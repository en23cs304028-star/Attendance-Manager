# 🎓 Attendance AI

A production-ready **facial recognition attendance management system** built with Streamlit, InsightFace, and Supabase. Designed as a Google Internship Project.

---

## ✨ Features

- **Student Registration** — Onboard students with a single photo; generates a 512-D ArcFace embedding stored in Supabase.
- **Video-Based Attendance** — Upload classroom/security footage; the system detects, aligns, and matches every face against the registered database automatically.
- **High-Accuracy ML Pipeline** — Uses affine warp alignment to normalise faces before embedding, eliminating geometric variance across frames.
- **Server-Side Vector Search** — Cosine similarity runs inside Postgres via `pgvector`, so no large embeddings are transferred over the network.
- **Session Management** — Create/end lecture sessions; attendance is bucketed into 5-second windows for a "Present / Partial / Absent" classification.
- **Unknown Face Resolution** — Manually assign unrecognised faces to registered students and retroactively update attendance.
- **CSV Export** — Download the final attendance report for any session.

## ⚙️ ML Pipeline (Summary)

1. **Detect** faces with InsightFace `buffalo_l`
2. **Gate** low-quality frames (size, boundary, blur, IoU checks)
3. **Align** via 5-point landmarks + `cv2.estimateAffinePartial2D` → 112×112 canonical crop
4. **Embed** with ArcFace → 512-D normalised vector
5. **Match** via Supabase `match_faces` RPC (cosine similarity ≥ threshold)

See [`ARCHITECTURE.md`](ARCHITECTURE.md) for full detail on every threshold and decision point.

## 🚀 Getting Started

### 1. Prerequisites

- Python 3.10+
- A [Supabase](https://supabase.com) project with `pgvector` enabled

### 2. Clone & Install

```bash
git clone https://github.com/<your-username>/attendance-ai.git
cd attendance-ai

python -m venv .venv
source .venv/bin/activate       # Windows: .venv\Scripts\activate

pip install -r requirements.txt
```

> **First run** will download the InsightFace `buffalo_l` model pack (~500 MB) automatically.

### 3. Configure Supabase

Copy the secrets template and fill in your credentials:

```bash
cp .streamlit/secrets.toml .streamlit/secrets.toml   # already there — just edit it
```

```toml
# .streamlit/secrets.toml
SUPABASE_URL = "https://<your-project-id>.supabase.co"
SUPABASE_KEY = "<your-anon-or-service-role-key>"
```

### 4. Set Up the Database

Run the following SQL in the Supabase SQL Editor:

```sql
-- Enable pgvector
CREATE EXTENSION IF NOT EXISTS vector;

-- Students registry
CREATE TABLE IF NOT EXISTS students (
    student_id   TEXT PRIMARY KEY,
    name         TEXT NOT NULL,
    roll_number  TEXT,
    department   TEXT,
    embedding    vector(512),
    created_at   TIMESTAMPTZ DEFAULT NOW()
);

-- Lecture sessions
CREATE TABLE IF NOT EXISTS lecture_sessions (
    id          BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    subject     TEXT,
    status      TEXT DEFAULT 'active',
    started_at  TIMESTAMPTZ DEFAULT NOW(),
    ended_at    TIMESTAMPTZ
);

-- Attendance logs
CREATE TABLE IF NOT EXISTS attendance_logs (
    id          BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    student_id  TEXT REFERENCES students(student_id),
    session_id  BIGINT REFERENCES lecture_sessions(id),
    status      TEXT DEFAULT 'Present',
    marked_at   TIMESTAMPTZ DEFAULT NOW()
);

-- Server-side vector matching RPC
CREATE OR REPLACE FUNCTION match_faces (
  query_embedding vector(512),
  match_threshold FLOAT,
  match_count     INT
)
RETURNS TABLE (student_id TEXT, name TEXT, similarity FLOAT)
LANGUAGE sql STABLE AS $$
  SELECT
    students.student_id,
    students.name,
    1 - (students.embedding <=> query_embedding) AS similarity
  FROM students
  WHERE 1 - (students.embedding <=> query_embedding) > match_threshold
  ORDER BY students.embedding <=> query_embedding ASC
  LIMIT match_count;
$$;
```

### 5. Run

```bash
streamlit run app.py
```

Open `http://localhost:8501` in your browser.

## 🔑 Key Thresholds

| Parameter | Default | Description |
|---|---|---|
| Min face size | 60×60 px | Ignores tiny background faces |
| Boundary margin | 10 px | Discards clipped faces at frame edges |
| Blur variance | 100.0 | Laplacian sharpness filter |
| IoU threshold | 0.3 | Spatial consistency check between frames |
| Match confidence | 0.65 | Cosine similarity for a positive match |
| Frame sampling | every 5th | Balances speed vs. coverage |
| "Present" cutoff | ≥ 60 % of windows | Core attendance criteria |
| "Partial" cutoff | ≥ 15 % of windows | Partial-credit criteria |
